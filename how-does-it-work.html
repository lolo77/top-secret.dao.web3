How it works ?

PNG images

PNG (Portable Network Graphics) is an image file format that uses a lossless compression algorithm.

Lossless means that each pixels remain exactely the same between two saves.

Red, Green, Blue and even Alpha components can store up to 8 bits.

Using the Least Significant Bit (LSB) of each pixel's component to store 1 bit at a time of the secret will not visually affect the image.

Even 2 or 3 bits can be done according to the image complexity. The higher the bits are changed, the most the visual impact will be.

This example shows a 100x100 pixels (white) 24-bits RGB image. It can hold up to 40,000 bytes.

JPEG images

Joint Photographic Experts Group is an image file format that is multilayered to optimize the compression of a photo-style image.

Compression algorithm (simplified) :

  • Layer 1 : Translate RGB to YCrCb - lossless
  • Layer 2 : Transform 8*8 pixels blocs to discrete cosine transform (DCT) coefs - lossy
  • Layer 3¬†: Huffman compression - lossless

The layer 2 is the heart of the problem : lossy part.

The data is then hidden into the least significant bits of each non-null DCT coef, instead of the pixel's components.

This example shows a 100x100 pixels (white) 24-bits RGB image. It can hold up to 565 bytes through the DCT coefficient bits.

The original file size is 3.10kB.

PDF documents

A Portable Document Format is a hierarchical structure with several maps.

A map is a key-value array. This is a good place to insert a special key and a part of secret.

This is quiet simple, fast and allow unlimited data to be hidden.
